Projects

MindBridge – Mental Health Support Platform

A responsive web app that connects users with licensed mental health professionals via chat and video sessions. Features include anonymous messaging, emergency support, and a blog offering coping strategies. Inspired by your passion to serve emotionally vulnerable users without needing physical visits. Tools: HTML, CSS, JavaScript, React, Firebase

AfroLens – Africa-Centric Storytelling Platform

A multimedia site that publishes spoken word pieces, historical articles, and animated short stories about African identity, struggle, and resilience. Visitors can listen, read, and comment on each piece. Built to combat cultural erasure and celebrate African voices. Tools: HTML, CSS, JavaScript, Tailwind CSS, Markdown integration

CodePulse – Developer Motivation Dashboard

A minimalist, personal dashboard for developers filled with motivational quotes, embedded tweets, learning milestones, and progress tracking. The dashboard updates dynamically and encourages consistency and mental well-being while learning to code. Tools: JavaScript, Local Storage, Twitter API, Bootstrap

ALX Resource Vault – Student Learning Companion

A curated learning portal for ALX students offering categorized tutorials, past assignments, cheat sheets, and productivity tools. Includes a search function and responsive design for mobile-first learning on the go Tools: HTML, CSS, JavaScript, Google Sheets API
